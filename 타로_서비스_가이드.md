**GitHub Pages 배포:**
1. GitHub 계정 생성
2. 새 Repository 생성 (public)
3. 두 파일 업로드
4. Settings → Pages → Source: main branch
5. 몇 분 후 https://username.github.io/repo-name 에서 접속 가능

### 2. 커스텀 도메인 연결

**도메인 구매:**

1. Domain settings → Add custom domain
2. 도메인 제공업체에서 DNS 설정
3. HTTPS 자동 적용 (Let's Encrypt)
